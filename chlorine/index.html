Installation Guide

Follow our step-by-step guide to install Chlorine Linux on your system:

  1. Download the ISO file from the link above
  2. Create a bootable USB drive using tools like Rufus, Etcher, or dd
  3. Boot from the USB drive
  4. From the boot menu, select your preferred installation method:
    • Graphical Installer - User-friendly installation with full graphical interface
    • Text-based Installer - Lightweight installation option for older hardware or server setups
  5. Follow the installation wizard to configure your system
  6. Enjoy your new Chlorine Linux system!

Special Boot Options

Chlorine Linux offers several specialized boot options:

  • Security Tools Mode - Boot with a comprehensive suite of security and penetration testing tools
  • Non-free Drivers - Include proprietary drivers for better hardware compatibility
  • Failsafe Mode - Boot with minimal settings for troubleshooting
